Возможности WesNal

Полный обзор всех функций лаунчера

Библиотека версий

Полный список доступных версий Minecraft с фильтрацией по типу: Vanilla, Fabric, Forge, Quilt, OptiFine. Встроенный поиск и пагинация. Выбор версии и запуск в один клик. Лаунчер автоматически устанавливает нужный загрузчик через @xmcl/installer.

Поиск модов (Modrinth)

Прямая интеграция с Modrinth API. Поиск по названию, фильтрация по категориям (моды, ресурспаки, шейдеры). Карточки с описанием, иконками и кнопками «Установить» и «Подробнее». Модальное окно с полной информацией и выбором версии.

WNO Core

Автоматическая сборка оптимизации одним кликом. Устанавливает Sodium, Lithium, Iris Shaders и другие моды для максимальной производительности. Конфигурация хранится в wno-core.json. Поддерживает выбор версии игры и загрузчика.

Живая консоль

Встроенный терминал с выводом логов Minecraft в реальном времени. Поддерживает команды: help, clear, gc, ram, version, java, ping. Цветовая подсветка ошибок и предупреждений.

Кастомизация интерфейса

12 категорий настроек: Launcher, Game, Display, Performance, Java, Assets, Accounts, Mods, Network, Social, Control, Experimental. Изменение темы, акцентного цвета, уровня размытия, анимаций и поведения лаунчера.

Microsoft-авторизация

Вход через настоящий Microsoft-аккаунт с использованием MSMC. Доступ к лицензионному скину, сессионным серверам и полной функциональности онлайн-игры.

Сборки (модпаки)

Создание и управление сборками модов. Возможность задать имя, версию Minecraft, добавлять моды через поиск. Экспорт в формат .wesnalpack и импорт готовых сборок.

Диагностика системы

Отображение информации о процессоре, ОЗУ, Java и ОС. Автоматические рекомендации по оптимизации. Кнопка «Оптимизировать» для подбора идеальных настроек ОЗУ и потоков.

Инструкция по установке

  1. Скачайте установщик WesNal-Setup.exe по кнопке выше.
  2. Запустите файл и выберите папку для установки лаунчера.
  3. Дождитесь завершения установки — лаунчер запустится автоматически.
  4. При первом запуске выберите версию Minecraft и нажмите «Играть».
  5. Для установки модов перейдите на вкладку «Дополнения» и воспользуйтесь поиском Modrinth.

    О проекте WesNal

    WesNal — современный Minecraft-лаунчер, созданный с упором на дизайн, производительность и удобство использования. Проект объединяет лучшие практики сообщества и передовые технологии для обеспечения максимального комфорта игроков.

    Разработчик

    Лаунчер разрабатывается независимым разработчиком, известным под никнеймом Prostynskiy. Связь через Telegram и Discord.

    Технологии

    Лаунчер построен на базе Electron с использованием Node.js для серверной логики и чистого HTML/CSS/JavaScript для интерфейса. Glassmorphism-дизайн достигается через CSS-свойства backdrop-filter и полупрозрачные поверхности. Для установки загрузчиков используется библиотека @xmcl/installer. Авторизация Microsoft реализована через MSMC.
